# TCC-Desenvolvimento-de-Sistemas
Este é um projeto colaborativo dos integrantes do curso de Desenvolvimento de Sistemas da Escola Caetano de Campos. O objetivo da criação dessa IA é o desenvolvimento das habilidades dos alunos constituintes do curso.

# Descrição do projeto
 
 Este projeto visa desenvolver uma IA assistente para auxiliar na escrita e otimização de códigos. 
 
 Principais funcionalidades:
 - Sugestões de melhorias com base no código existente.
 - Geração de código a partir do conhecimento fornecido pelo usuário.
 - Aplicativo mobile para edição e aprimoramento remoto do código.
 - Análise de vídeos de gameplay para sugerir ajustes na movimentação e câmera de jogos.
 
 A IA será capaz de entender e propor soluções baseadas no contexto do código, oferecendo uma experiência intuitiva e eficiente para desenvolvedores.
